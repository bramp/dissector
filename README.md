dissector
=========

Dissects various file formats. Think wireshark but for files.

Provides a API, command line app, and GUI.

Supports
--------
* MBR - Master Boot Record
* ISO files - Media format (that is used by mp4 and others)
* Java class files
* PNG files
* Torrent files
* Zip files
